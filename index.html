The software being used is an extension of the Brat annotation tool developed by researchers at the University of Tokyo. We have extended the tool to allow the saving and direct export of RDF to a specified Triplestore - information about our linked data fork can be found on google-code.

The ontology used with the RDF is OntoMedia, developed by Drs K Faith Lawrence, Michael Jewell and Mischa Tuffield. The OWL description of the ontology are currently hosted in google-code.

Discussion group: Digital Narrative (at Google Groups)

The annotations are browseable but an account is required to edit or create annotations. If you would like to request an account then contact Faith Lawrence (faith dot lawrence at kcl dot ac dot uk)

We will be running a second workshop at DRHA (Greenwich, London. 31st Aug - 3 Sept)

DH2014: Linked Data and Literature: Encoding the Facts in Fiction
